# **45-Day DSA Roadmap (Beginner → Advanced)**

Designed for 1–2 hours per day.

# **PHASE 1: BEGINNER (Day 1–15)**

Focus: Foundations, pattern recognition, problem-solving basics.

## **Week 1: Basics + Arrays**

### **Day 1:** Time & Space Complexity

• Learn Big-O basics

• Exercise: Analyze complexity of 5–6 simple code snippets

• Resource: Abdul Bari Time Complexity (YouTube)

### **Day 2:** Array Basics

• Static vs Dynamic arrays

• Exercise: Insert, delete, search

• Resource: NeetCode Arrays playlist

### **Day 3:** Prefix Sum

• Exercise: Subarray sum queries

• Problem: "Range Sum" (LeetCode)

### **Day 4:** Sliding Window Basics

• Fixed window

• Problem: Max sum subarray of size K

### **Day 5:** Two Pointer Technique

• Problems: Pair sum, remove duplicates

### **Day 6:** Searching (Binary Search)

• Standard + search in rotated array

• Problem: Binary Search, First/Last occurrence

### **Day 7:** Mini-Project

**Array Utility Tool**
• Build a small CLI tool that performs:

– Insert

– Delete

– Search

– Prefix sum

– Max subarray


## **Week 2: Strings + Hashing**

### **Day 8:** String Basics

• Exercise: Palindrome, reverse, frequency count

### **Day 9:** Hashmaps

• Problem: Two Sum, Anagram

### **Day 10:** Sliding Window on Strings

• Problem: Longest substring without repeat

### **Day 11:** String Pattern Problems

• Problem: Valid Parentheses, Isomorphic Strings

### **Day 12:** Sorting

• Bubble, Selection, Insertion (concept only)

• Focus on practical sorts: Merge, Quick

• Problem: Sort colors

### **Day 13:** Recursion Basics

• Factorial, Fibonacci, subsets

### **Day 14:** Mini-Project

**String Analyzer App**
• Features:
– Count characters

– Find substrings

– Check anagram

– Sliding-window insights

### **Day 15:** Beginner Mock Test

Solve 10 beginner DSA problems.

# **PHASE 2: INTERMEDIATE (Day 16–30)**

Focus: Trees, Graphs & DP foundations.


## **Week 3: Linked Lists + Stacks + Queues**

### **Day 16:** Linked List Basics

• Insert, delete, reverse

### **Day 17:** Fast & slow pointer

• Problem: Cycle detection, middle element

### **Day 18:** Stack

• Problems: Next Greater Element, Min Stack

### **Day 19:** Queue + Deque

• Problems: Sliding Window Maximum

### **Day 20:** Priority Queue (Heap)

• Problems: K largest elements

### **Day 21:** Mini-Project

**Browser History Simulator**
Using: Stack + Queue + Linked List

## **Week 4: Trees + Graphs**

### **Day 22:** Binary Tree Basics

• DFS (Inorder, Preorder, Postorder)

### **Day 23:** BFS / Level Order

• Problems: Level order traversal

### **Day 24:** Binary Search Trees

• Insert, search, delete in BST

### **Day 25:** Graph Basics

• Representation: adjacency list
• BFS, DFS

### **Day 26:** Graph Cycle Detection

• Directed + undirected

### **Day 27:** Topological Sort + Shortest Paths

• Dijkstra + BFS shortest path

### **Day 28:** Mini-Project

**Path Finder App (Graph Project)**
• Input: Graph nodes and edges
• Output: DFS path, BFS path, shortest path

### **Day 29–30:** Intermediate Test

Solve 15 mixed problems.

# **PHASE 3: ADVANCED (Day 31–45)**

Focus: Dynamic Programming + Advanced Graphs + Tricky Problems.

## **Week 5: Dynamic Programming**

### **Day 31:** DP Basics

• Memoization + Tabulation
• Problem: Climbing Stairs

### **Day 32:** Knapsack

• 0/1, subset sum

### **Day 33:** DP on Strings

• LCS

• Edit Distance

### **Day 34:** DP on Grids

• Unique paths

• Minimum path sum

### **Day 35:** DP on LIS + Bitmask DP (Intro)

### **Day 36:** Mini-Project

**DP Visualizer**
• Show recursion tree
• Convert to DP
• Compare execution time

## **Week 6: Advanced Graphs + Final Prep**

### **Day 37:** Union Find (DSU)

• Problems: Connected components

### **Day 38:** Minimum Spanning Tree

• Kruskal + Prim

### **Day 39:** Trie

• Insert, search, autocomplete

### **Day 40:** Backtracking

• N-Queens

• Subsets

• Permutations

### **Day 41:** Bit Manipulation

• Count set bits

• Single number problems

### **Day 42–44:** Final Revision + Mock Tests

• Solve 25 problems from mixed topics

• Try 3 timed contests on LeetCode

Pick one:
